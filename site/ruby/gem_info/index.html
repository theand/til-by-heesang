콘솔에서 rubygems 의 특정 gem 상세 정보 보기

gem info 커맨드가 2018-12에 릴리즈된 3.0.0 에서야 들어왔는데, npm info 처럼 수시로 사용하기엔 턱없이 부족한 것 같다.

#로컬에 설치된 패키지 정보 보여주기
gem info name

#원격에  있는 패키지 정보 보기
gem info name -r
